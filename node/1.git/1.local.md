## 添加到暂存区
```
git add ./ -A / 文件名
```
## 添加到历史区
```
git commit -m '消息'
```
## 查看版本信息
```
git log --grep / --author
```

## git diff
- git diff 比较工作区和暂存区
- git diff --cached 比较暂存区和历史区的不同
- git diff master 比较工作区和历史区的不同
